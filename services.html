services that Plus St Hosts.

From online messaging to metasearch-engines, we have it all.
Privately. Securely. Fastly. Seriously.

Private Bin

Private bin is a easy to use and minimal pastebin with password lock, one time viewing and encryption.

Libreddit

Libreddit is a sleek, modern, simple and private reddit viewer.

Jitsi

Jitsi is a private, secure, and open source video conferencing solution with many features such as encryption and no need for sign ups.

Mumble

Mumble is an open source, high quality, and low latency voice chat applicaton. You can access our mumble server at mumble.plus.st with the port 64738.

Akkoma

Akkoma is a decentralized social network with no algorithms and advertisments which puts you in control.

Bitwarden

Bitwarden is a simple, private, secure, open source, and cloud focused password manager. (powered by vaultwarden).

Matrix

Matrix is a federated, private, secure, and feature rich network for communication.

Gitea

Gitea is a fast and easy to use git server which makes managing all of your git repositories simple.
